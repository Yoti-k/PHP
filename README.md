# Описание классов проекта

В данном проекте реализована базовая структура пользователей с использованием наследования в PHP. Проект состоит из двух основных классов, находящихся в пространстве имен `MyProject\Classes`.

## 1. Класс User
Базовый класс, описывающий стандартного пользователя.

**Расположение:** `src/Classes/User.php`

**Свойства:**
* `public $name` — Имя пользователя.
* `public $login` — Логин для входа.
* `private $password` — Пароль (скрытое свойство).

**Методы:**
* `__construct($name, $login, $password)` — Инициализирует свойства объекта при создании.
* `showInfo()` — Возвращает HTML-блок с информацией о пользователе (Имя, Логин).
* `__destruct()` — Выводит сообщение об удалении объекта (пользователя) при завершении скрипта.

---

## 2. Класс SuperUser
Класс расширяет функциональность `User` и предназначен для администраторов или пользователей с особыми правами.

**Расположение:** `src/Classes/SuperUser.php`
**Родительский класс:** `User`

**Свойства:**
* `public $role` — Роль пользователя (например, "admin" или "moderator").

**Методы:**
* `__construct($name, $login, $password, $role)` — Перегружает конструктор родителя. Вызывает родительский конструктор для установки основных данных и дополнительно устанавливает свойство `$role`.
* `showInfo()` — Переопределяет метод родителя. Возвращает HTML-блок, содержащий Имя, Логин и **Роль** суперпользователя.
